# Meta Interview Style Notes

## Coding
- Meta focuses on practical coding — you'll be in a shared doc or CoderPad
- Communication is critical — narrate everything
- Multiple short problems (2-3 per session) rather than one deep problem
- Expect follow-ups that extend the problem

## Behavioral
- Meta values execution and speed
- Questions focus on impact, ownership, and results
- STAR format heavily weighted

## Difficulty Calibration
- Junior: LeetCode Easy-Medium
- Mid/Senior: LeetCode Medium
- Staff: LeetCode Medium-Hard
