# Google Interview Style Notes

## Coding
- Google emphasizes optimal solutions with clear complexity analysis
- Interviewers often ask "Can you do better?" even for O(n) solutions
- Expect follow-ups that add constraints (e.g., "now do it in one pass")
- System design rounds focus on scale and reliability

## Behavioral
- Google values "Googleyness" — cultural fit, ambiguity handling, collaboration
- Leadership (even for IC roles) is a recurring theme
- Expect questions about dealing with ambiguity

## Difficulty Calibration
- Junior: LeetCode Medium
- Mid/Senior: LeetCode Medium-Hard
- Staff: LeetCode Hard + system design deep dive
